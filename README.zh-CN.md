## 主要特性

* 询问 Claude：选择任意文本并右键点击即可获得 AI 帮助
* 代码文档：自动生成代码文档
* 上下文感知：可包含之前回复的上下文
* Markdown 输出：回复采用清晰易读的 Markdown 格式

## 快速入门

1. 安装扩展
2. 设置您的 Claude API 密钥（见下文）
3. 选择文本，右键点击，选择 Claude AI，然后选择：
   * `询问 Claude` 获取常规帮助
   * `生成文档` 自动生成文档

就是这么简单！

## API 密钥设置

### 获取密钥
1. 注册 Anthropic 账户
2. 进入 API 设置
3. 生成新的 API 密钥
4. 确保安全 - 切勿分享或提交密钥

### 添加密钥

方法 1 - VS Code 设置（推荐）：
1. 打开设置（Ctrl/Cmd + ,）
2. 搜索 "Claude VS Code"
3. 输入您的 API 密钥
4. VS Code 将安全存储密钥

方法 2 - 环境变量：
* 在环境中设置 CLAUDE_API_KEY
* 确保安全的变量管理

## 安全第一

* 密钥存储在 VS Code 的安全存储中
* 仅使用 HTTPS API 通信
* 无数据存储或日志记录
* 直接 Claude API 集成
* 您的密钥，您做主

## 模型选择

在 VS Code 设置中选择您的模型：
* claude-3-opus-20240229（默认）
* claude-3-sonnet-20240229
